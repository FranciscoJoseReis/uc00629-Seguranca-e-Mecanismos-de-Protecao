# 🔐 Segurança e Mecanismos de Proteção

## 📌 Descrição

Este projeto consiste num **relatório técnico interativo em formato web**, desenvolvido em HTML5, que demonstra a implementação de mecanismos de segurança em sistemas Windows.

O site apresenta procedimentos práticos acompanhados por vídeos explicativos, incluindo:

* Configuração da Firewall
* Gestão de Antivírus
* Políticas de Segurança Local
* Encriptação de dados com BitLocker

---

## 🎯 Objetivo

Demonstrar, de forma clara e prática, como configurar e reforçar a segurança de um sistema operativo Windows, utilizando uma máquina virtual.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Vídeo (MP4)
* VirtualBox (ambiente de testes)

## 🔎 Funcionalidades

### 🔥 Firewall

* Criação de regra para bloquear a porta 23 (Telnet)

### 🛡️ Antivírus

* Verificação de ameaças com Windows Defender
* Análise complementar com Malwarebytes

### 👤 Políticas de Segurança

* Configuração de palavras-passe seguras
* Bloqueio de conta após tentativas falhadas

### 🔐 Criptografia

* Encriptação de disco com BitLocker
